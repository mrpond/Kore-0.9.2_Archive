This is the official homepage of the Kore Project - An open source, complete console-based Ragnarok client. Gravity (company who makes Ragnarok Online) has nothing to do with Kore, this project is unofficial.


What is Kore?

Kore is a console-based Ragnarok client/bot, written in Perl.

Kore is designed to attack on its own, and be a party helper.

Note that this client uses only the primitive console - not an ideal interface, but good enough.


How do I install Kore?

Kore can be run on Windows and Linux, and if requested, versions can be compiled for Mac operating systems.

Kore is run like any other program. Grab Kore from the download page, unzip, and execute. Head over to the Help section for console commands, chat commands (receieve commands in-game!), and configuration options.


How do I compile Kore?

Windows

Download and install ActivePerl for Windows

Download and install Perl2exe for Windows

Next you must install the necessary Perl packages:

1) Go to Start->Run... and run 'ppm'
2) In the console type 'install Time::HiRes'

In the Kore source directory make a file called "Kore.bat". In this file put the following line:

perl Kore.pl

To run Kore without compiling double-click on Kore.bat.


In the Kore source directory make a file called "Kore-compile.bat". In this file put the following line:

perl2exe -o=Kore.exe Kore.pl

To compile Kore just double-click on Kore-compile.bat.



Linux

Install your distribution's version of Perl 5.6.1, and use cpan to install Time::HiRes.

Download and install Perl2exe for Linux.

To run without compiling, at command line switch to the Kore source directory and type:

perl Kore.pl


To compile, at command line switch to the Kore source directory and type:

perl2exe -o=Kore Kore.pl
