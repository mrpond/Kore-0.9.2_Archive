★追加コマンド

exp [reset]
  時給の表示。
  resetオプションを付けるとリセットします。

ignoreall (flag)
  1に設定すると全ての耳打ちを拒否します。

★追加設定

wait_ReConnection
  再接続時のウェイトを設定します。
  0〜60の範囲内にすること。

waitDetection
  ターゲット被り検知時の待機秒数を指定します。

useDetection
  ターゲット被り検知機能を使うかどうかを設定します。
  0 : OFF.
  1 : sorryword.txtからランダムに発言。
  2 : useSorryEmotionのエモーションを出す。

useSorryEmotion
  useDetectionが2の時のエモーション番号を指定します。

useAngry
  横殴りをされたときの動作を設定します。
  0 : OFF.
  1 : angryword.txtからランダムに発言。
  2 : useAngryEmotionのエモーションを出す。

useAngryEmotion
  useAngryが2の時のエモーション番号を指定します。

distResponse
  ここで指定した範囲内のキャラクターの発言に対して自動返答します。

itemsImportantAuto
  重要アイテム取得ルーチンを使うかどうかを設定します。

timeCutImportant
  重要アイテム取得ルーチン作動時、ここで指定した秒数だけ攻撃を停止します。

distImportantItem
  ここで指定した範囲内のアイテムに対して重要アイテム取得ルーチンを発動します。

recordPlayerInfo
  ユーザー情報を記録するかを設定します。

recordEmotion
  エモーションをログに記録するかを設定します。

avoidPortal
  ワープポータルを回避するかを設定します。

shareOnMapChanged
  マップチェンジ毎に経験値公平モードにするかを設定します。

useSelf_item_continue
  自動アイテム使用が発動された時に、ここで指定した回数だけ連続使用します。

useThanks
  ヒール等をかけて貰った時に返答するかを設定します。
  0 : OFF.
  1 : useThanksWordを発言する。
  2 : useThanksEmotionのエモーションを出す。

useThanksWord
  useThanksが1の時の発言を設定します。

useThanksEmotion
  useThanksが2の時のエモーションを指定します。

hprOnStart
  Kore開始時にHP回復スキルLv1を覚えるパケットを送ります。

sprOnStart
  Kore開始時にSP回復スキルLv1を覚えるパケットを送ります。
